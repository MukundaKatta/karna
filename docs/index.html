Key Features

Multi-Channel Messaging

Slack, Discord, WhatsApp, SMS, email, and custom integrations from a single platform.

🧩

Extensible Skills

Plugin architecture for adding new capabilities. Build and share custom skills.

🧠

Semantic Memory

Long-term memory with vector search for contextual, personalized conversations.

🔒

Self-Hosted

Deploy on your own infrastructure. Full data ownership, complete customization.

🔄

Conversation Threading

Maintains context across multi-turn conversations for natural interactions.

🔐

Role-Based Access

Fine-grained permissions for secure team deployments and collaboration.

Quick Start

# Clone the repository
git clone https://github.com/MukundaKatta/karna.git
cd karna

# Configure environment
cp .env.example .env

# Launch with Docker
docker-compose up -d

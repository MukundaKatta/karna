# @karna/mobile

Native mobile app for Karna on iOS and Android, built with Expo.

## Features

- Chat with your Karna agent on the go
- Push notifications for agent responses
- Haptic feedback and native gestures
- Audio/video message support
- Clipboard integration

## Setup

```bash
pnpm install
```

## Usage

```bash
pnpm start     # Start Expo dev server
pnpm ios       # Run on iOS simulator
pnpm android   # Run on Android emulator
pnpm web       # Run in browser
```

## Tech Stack

- **Expo ~52** — React Native framework
- **Expo Router** — File-based navigation
- **React Native 0.76** — Native UI
- **Zustand** — State management
- **Reanimated** — Animations

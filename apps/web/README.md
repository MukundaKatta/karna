# @karna/web

Web dashboard for managing Karna agents, skills, and conversations.

## Features

- Chat interface for agent interaction
- Agent management dashboard
- Skill configuration
- Session history viewer
- Analytics and usage charts
- Settings management

## Setup

```bash
pnpm install
pnpm dev    # Start on http://localhost:3000
```

## Build

```bash
pnpm build
pnpm start
```

## Tech Stack

- **Next.js 15** — React framework
- **React 19** — UI library
- **Tailwind CSS 4** — Styling
- **Zustand** — State management
- **Recharts** — Charts and analytics
- **Lucide React** — Icons
